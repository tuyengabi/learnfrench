Comprenez ce qu'est le DOM

Bonjour , et bienvenu dans ce cours . Ou nous allons apprendre à écrire du Js pour le web. Pour accéder aux élément d'une page web et les modifier avec le langage Js : une interface de programmation existe : le DOM.


Dans ce chapitre , nous allons voir ce qui se cache sous l'acronyme DOM. Nous découvrirons comment il est structuré , à quoi il sert, et quelle est sa relation avec HTMl.



Accédez aux élément du DOM

Le DOM permet de modifier des éléments d'une page web avec le langage Js.Mais comment accéder aux élément que l'on souhait modifier? C'est ce que nous allons voir dans ce chapitre. Le point de départ du DOM , c'est le document. Il représente votre page : c'est lui qui contient les fonctions dont vous aurez besoin pour rechercher des éléments dans votre page. Et pour retrouver ces éléments, plusieurs méthodes sont possibles. Nous allons decouvrir quelques-unes dans ce chapitre.


Tout d'abord, nous verrons comment rechercher in élément grace à son ID. Puis , nous découvrirons une méthode qui offre la possibilité de faire une recherche grace à la classe d'un élément. Ensuite , nous apprendrons à rechercher tous les éléments du DOM avec un nom de balise bien précis. Pour finir, nous découvrirons une recherche qui met en oeuvre plusieurs procédés. Elle se sert pour cela de la puissance des sélecteur CSS. Sachez également qu'il n'y a pas qu'avec Documents que l'on peut rechercher un élément. Il existe dse fonction qui permettent de rechercher les enfants et les parents d'un élément.
